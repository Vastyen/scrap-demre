# Install

## Requirements
* Python3
* Python Installer Package
* Pandas
* Request
* BeautifulSoup
* Sqlite3

## Linux

          sudo apt-get update

          cd script
          git clone https://github.com/Bescri/scrap-demre
          cd scrapt-demre/
